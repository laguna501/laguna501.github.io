VO_OSMP_RETURN_CODE com.visualon.OSMPPlayer.VOCommonPlayerHTTPConfiguration.setHTTPRetryTimeout ( int  time)

Set HTTP connection retry timeout.

The setting is applicable only after data source has been opened successfully.

Parameters
time[in] interval, in seconds, in which the player will attempt to re-establish the HTTP connection. The default is 120 seconds. Setting the value to -1 disables the timeout so that the player will keep retrying, until the connection is established again
Returns
VO_OSMP_RETURN_CODE#VO_OSMP_ERR_NONE if successful
