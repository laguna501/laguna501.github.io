VO_OSMP_RETURN_CODE com.visualon.OSMPPlayer.VOCommonPlayerControl.setSurfaceChangeFinished ( )

Notify the player that surface changed event has been completed and video rendering can be started.

This method is called when receiving SurfaceHolder.SurfaceChanged(). When surface is changing, video rendering is suspended until this API is called. It will time out after 200ms if the API is not called, and video rendering will continue.

Returns
VO_OSMP_RETURN_CODE#VO_OSMP_ERR_NONE if successful.
