- (VO_OSMP_RETURN_CODE) clearSelection

Remove all current uncommitted selections.

Returns
VO_OSMP_ERR_NONE if successful
- (VO_OSMP_RETURN_CODE) commitSelection

Commit all current asset selections.

If any asset type is not selected, current playing asset of that type is used. This operation will remove all current selections after commit.

Returns
VO_OSMP_ERR_NONE if successful
- (int) getAudioCount

Get the number of available audio tracks.

An audio track is defined as a combination of language and codec type. An audio track may have multiple audio bit rates. The final playback bit rate depends on selected video track.

Returns
number of available audio tracks; -1 if unsuccessful.

Returns

- (id<VOOSMPAssetIndex>) getCurrentSelection

Get the current track selection of video/audio/subtitle.

If a track is selected automatically, then the corresponding index in VOOSMPAssetIndex is set to VO_OSMP_ASSET_AUTO_SELECTED. If a track is being manually selected, its index is returned through VOOSMPAssetIndex. If no track of a specific type is being selected automatically/manually, then the corresponding index in VOOSMPAssetIndex is set to VO_OSMP_ASSET_NOT_SELECTED.

Returns
a VOOSMPAssetIndex object if successful; nil if unsuccessful. Returned object holds the current index values of the selected video, audio, and subtitle tracks.
- (id<VOOSMPAssetIndex>) getPlayingAsset

Get the actual indices of video/audio/subtitle tracks which are being played.

If a track is being played, its index is returned through VOOSMPAssetIndex. If no track of a specific type (mostly likely subtitle) is being played, then the corresponding index in VOOSMPAssetIndex is set to VO_OSMP_ASSET_NOT_SELECTED.

Returns
a VOOSMPAssetIndex object if successful; nil if unsuccessful. Returns the actual indices of video/audio/subtitle tracks which are being played.

- (VO_OSMP_RETURN_CODE) setPreferredAudioLanguage: (NSArray *)  languageList

Set preferred audio language.

Parameters
languageList[in] a list of preferred specified using ISO 639-2 code. For example, "fre" stands for French. Based on the order of languages specified in the list, the player will select the first available language in the stream. If the list is empty, the player will select it based on the streaming specification.
Returns
VO_OSMP_ERR_NONE if successful
