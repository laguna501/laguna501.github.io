- (VO_OSMP_RETURN_CODE) open: (NSString *)  url
flag: (VO_OSMP_SRC_FLAG)  flag
sourceType: (VO_OSMP_SRC_FORMAT)  sourceType
initParam: (VOOSMPInitParam *)  initParam
initParamFlag: (int)  initParamFlag 

Open media source.

Parameters
url[in] Source file description (e.g. an URL or a file descriptor, etc.)
flag[in] The flag for opening media source. Refer to VO_OSMP_SRC_FLAG.
type[in] Indicates the source format. Refer to VO_OSMP_SRC_FORMAT. Default value is VO_OSMP_SRC_AUTO_DETECT
initParam[in] Initial parameters. Refer to VOOSMPInitParam. Valid fields depend on initParamFlag. If (initParamFlag & VO_OSMP_FLAG_INIT_ACTUAL_FILE_SIZE) > 0, You should set the fileSize.
initParamFlag[in] InitParam flag(s). Refer to VO_OSMP_INIT_PARAM_FLAG. You can use "|" (or) to input several parameters.
Returns
VO_OSMP_ERR_NONE if successful
Deprecated:
this function is replaced by VOCommonPlayerControl#open:flag:sourceType:openParam: and will be removed in the future
