VOOSMPAssetIndex com.visualon.OSMPPlayer.VOCommonPlayerAssetSelection.getCurrentSelection ( )

Retrieve the current track selection of video/audio/subtitle.

If a track is selected automatically, then the corresponding index in VOOSMPAssetIndex is set to VO_OSMP_ASSET_AUTO_SELECTED. If a track is being manually selected, its index is returned through VOOSMPAssetIndex. If no track of a specific type is being selected automatically/manually, then the corresponding index in VOOSMPAssetIndex is set to VO_OSMP_ASSET_NOT_SELECTED.

Returns
A VOOSMPAssetIndex object if successful or null if failed. Returned object holds the current index values of the selected video, audio, and subtitle tracks.

Implemented in com.visualon.VOOSMPStreamingDownloaderImpl.VOOSMPStreamingDownloaderImpl.

VOOSMPAssetIndex com.visualon.OSMPPlayer.VOCommonPlayerAssetSelection.getPlayingAsset ( )

Retrieve the actual indices of video/audio/subtitle tracks which are being played.

If a track is being played, its index is returned through VOOSMPAssetIndex. If no track of a specific type (mostly likely subtitle) is being played, then the corresponding index in VOOSMPAssetIndex is set to VO_OSMP_ASSET_NOT_SELECTED.

Returns
A VOOSMPAssetIndex object if successful or null if failed. Returned the actual indices of video/audio/subtitle tracks which are being played.

Implemented in com.visualon.VOOSMPStreamingDownloaderImpl.VOOSMPStreamingDownloaderImpl.

VO_OSMP_RETURN_CODE com.visualon.OSMPPlayer.VOCommonPlayerAssetSelection.setPreferredAudioLanguage ( String[]  languageList)

Set preferred audio language.

Parameters
languageList[in] a list of preferred specified using ISO 639-2 code. For example, "fre" stands for French. Based on the order of languages specified in the list, the player will select the first available language in the stream. If the list is empty, the player will select it based on the streaming specification.
Returns
VO_OSMP_ERR_NONE if successful
